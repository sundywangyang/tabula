使用方式

主稿：build-assets/icon/icon.svg

导出脚本：build-assets/icon/export.sh（需 rsvg-convert + ImageMagick + macOS iconutil）

Electron 配置（apps/main/src/main/index.ts）：

new BrowserWindow({
  icon: path.join(__dirname, '../../build-assets/icon/512.png'),
  // 或 macOS: 'build-assets/icon/Tabula.icns'
  //    Windows: 'build-assets/icon/Tabula.ico'
})
